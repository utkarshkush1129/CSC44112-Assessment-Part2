# CSC-44112 Part 2 – Iris Species Classification

## Project Overview
Multi-class flower species classification using the Iris dataset and a tuned Random Forest classifier.  
**Final Test Accuracy: 96.67% | Macro F1: 0.97**

---

## Dataset
- **Source:** UCI Machine Learning Repository — https://archive.ics.uci.edu/dataset/53/iris
- **Samples:** 150 (50 per class, perfectly balanced)
- **Features:** Sepal length, Sepal width, Petal length, Petal width (all in cm)
- **Target:** Species — setosa, versicolor, virginica
- **Missing values:** None
